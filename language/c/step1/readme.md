
# compile 

To compile `server.c`

```
gcc server.c -o server
```

To compile `client.c`

```
gcc client.c -o client
```

# command

To run server 
 - Port : TCP 5000

```
$ server 
```

To run client

```
$ client $IP
 Usage: client <ip of server> 
```
